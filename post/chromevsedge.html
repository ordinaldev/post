Microsoft Edge Vs Google Chrome

Chrome atau Edge
Mana yang lebih baik ?

Pengenalan

Microsoft Edge dan Google Chrome merupakan dua browser terkenal yang marak digunakan saat ini. Microsoft edge dikembangkan oleh Microsoft dan Google chrome dikembangkan oleh Google. walaupun berbeda pengembang tetapi kedua browser ini sama sama berbasis chromium dan V8. Chromium adalah shell browser, sedangkan V8 adalah mesin JavaScript yang memproses sebagian besar kode yang ditulis pengembang untuk aplikasi web.


Keunggulan

keunggulan dari kedua browser ini lebih dominan dari sisi Microsoft edge seperti :


1. tehubung dengan cortana

Cortana adalah asisten pribadi berbasis Artificial Intelligence (AI) yang dikembangkan oleh Microsoft yang membantu pengguna Windows 10 dalam banyak hal. Integrasi Cortana di Microsoft Edge memungkinkan pengguna untuk mencari apa pun tanpa mengetikkan kata kunci yang diperlukan.


2. Lebih ringan.

Microsoft Edge dikembangkan dengan tujuan menjadi browser yang lebih ringan dan lebih mumpuni dibandingkan browser lainnya. hal ini sering kali menjadi kontroversi dikarenakan sebagian user tidak berpendapat demikian. saya pun melakukan riset tentang ini dan menemukan hasil bahwa memang microsoft edge lebih ringan dari pada chrome. tetapi microsoft edge menjadi berat dikarenakan Microsoft yang terus terusan menambahkan fitur yang bahkan tidak pengguna butuhkan yang membuat browser ini menjadi berat sehingga banyak fitur yang harus dimatikan untuk membuat browser ini lebih ringan


3. Fitur .

Keunggulan lainnya yaitu fitur yang dimiliki Microsoft Edge namun tidak dimiliki Chrome. fitur tersebut seperti Catatan, Pembaca Immersive, Cortana, dan banyak lagi. ini menjadi kelebihan dan kekurangan dari edge. kekurangannya seperti fitur Shopping di Microsoft edge yang kini sering dianggap spam



Kesimpulan

Baik google chrome atau microsoft edge memiliki kelebihan dan kekurangannya masing masing.tergantung spesifikasi dari perangkat yang dimiliki. meskipun memang Microsoft Edge memiliki kelebihan dalam hal pengguna resource yang lebih rendah, sayangnya dengan banyaknya fitur yang ditambahkan oleh Microsoft, itu justru menambah beban resource untuk browser itu sendiri. dan untuk itu saya merekomendasikan Google Chrome sebagai browser default.


Google Chrome dan Microsoft Edge keduanya memiliki versi Android dan iOS.Namun, Edge tidak terlalu populer di perangkat seluler.Perbedaannya mungkin dapat dijelaskan oleh fakta bahwa Edge adalah browser default di komputer Windows, sedangkan Chrome adalah browser default di perangkat Android. Secara keseluruhan, kualitas aplikasinya serupa, dan pilihannya tergantung pada Selera pengguna.
